# Frontend - Contact App

Frontend de una aplicación de contacto desarrollada con React + Vite.  
Permite enviar mensajes a través de un formulario que se conecta con una API propia.
PROYECTO EN EL CUAL SE VE MI CURVA DE APRENDIZAJE :)

---

## 🚀 Tecnologías

- React
- Vite
- TailwindCSS
- DaisyUI
- Axios
- React Router

---

## 📦 Instalación

Clonar el repositorio:

git clone <tu-repo-url>

Entrar en la carpeta del proyecto:

cd nombre-del-proyecto

Instalar dependencias:

npm install

---

## ⚙️ Variables de entorno

Crear un archivo `.env` en la raíz del proyecto:

VITE_API_URL=http://localhost:4000/api/contact

> Ajustar la URL según el entorno (local o producción).

---

## ▶️ Ejecutar en desarrollo

npm run dev

---

## 🏗️ Build para producción

npm run build

---

## 📌 Notas

- El backend se encuentra en un repositorio separado.
- El formulario valida los campos antes de enviar los datos.
- El proyecto está preparado para deploy en Vercel, Netlify o similar.
- En el futuro se agregara nuevas secciones, como un admin panel and user
